Dogear
読んだページを記憶する

Chrome の標準履歴はタイトルと URL しか保存しません。
Dogear は「実際に読んだ段落」を自動記録し、あとから内容で全文検索できます。

Chrome に追加（無料） GitHub で見る

「あの記事、なんだっけ…」を解決する

読んだはずの内容が思い出せない。Chrome の履歴を検索しても、タイトルや URL でしか探せない。Dogear はその問題を解決します。

Chrome 標準履歴
  • タイトル・URL しか保存しない
  • 内容で検索できない
  • 分析・可視化ができない
  • 読んだかどうか分からない
Dogear
  • 読んだ段落テキストを保存
  • 内容でフルテキスト検索
  • ジャンル・時間帯・ドメイン分析
  • 滞在時間で「本当に読んだ」を判定
Features

主な機能

👁️

自動段落検出

IntersectionObserver と滞在タイマーを組み合わせ、スクロールしただけの要素は無視。本当に目を通した段落だけを記録します。

🔍

全文検索

タイトル・URL に加え、記録した段落テキストでも検索できます。「あの記事で読んだキーワード」で即座に見つけられます。

🏷️

ジャンル自動分類

ニュース・テクノロジー・ショッピング・SNS・動画・調査など 8 ジャンルに自動分類。フィルターで絞り込んで探せます。

📊

データ分析

KPI カード・ジャンル棒グラフ・時間帯ヒートマップ・ドメインランキングで、自分のブラウジングパターンを可視化します。

🔒

プライバシー設計

銀行・医療・保険などのサイトはデフォルトで除外。追加の除外ドメインも設定可能。データは端末内にのみ保存。

📤

JSON / CSV エクスポート

収集したデータはいつでも JSON または CSV 形式でダウンロードできます。データはあなたのものです。

プライバシーファースト

Dogear は外部サーバーへのデータ送信を一切行いません。すべてのデータはお使いの端末上の IndexedDB にのみ保存されます。

✓完全ローカル保存
✓外部送信ゼロ
✓金融・医療サイト除外
✓カスタム除外ドメイン対応

インストール方法

Chrome ウェブストアからワンクリックで追加できます

1

Chrome ウェブストアのページを開く

2

「Chrome に追加」をクリック

3

普通に Web を閲覧するだけで自動記録開始
