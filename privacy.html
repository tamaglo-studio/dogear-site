1.拡張機能の概要

Dogear（以下「本拡張機能」）は、ユーザーが Web ページを閲覧する際に「実際に読んだ段落テキスト」を自動検出・記録し、あとからキーワードで全文検索できるようにする Chrome 拡張機能です。

本拡張機能の唯一の目的は、ユーザー自身の閲覧内容を端末ローカルに記録・検索・分析することです。収集したデータは外部サーバーへ一切送信されません。

2.収集するデータ

本拡張機能が収集するデータは以下のとおりです。

データの種類具体的な内容収集
ウェブ履歴 訪問した URL・ページタイトル・訪問日時・滞在時間・ドメイン 収集する
ウェブサイトのコンテンツ 一定時間（デフォルト3秒）以上ビューポートに表示された段落テキスト 収集する
ユーザーのアクティビティ スクロールによる要素の表示状態（IntersectionObserver）・ページ滞在時間 収集する

✓ 上記データはすべてユーザーの端末内（IndexedDB）にのみ保存されます。外部への送信は行いません。

3.収集しないデータ

データの種類収集
個人を特定できる情報（氏名、住所、メールアドレスなど）収集しない
健康・医療に関する情報収集しない
財務・支払いに関する情報収集しない
認証情報（パスワード、PIN など）収集しない
個人的なコミュニケーション（メール、チャットなど）収集しない
位置情報（GPS、IPアドレスなど）収集しない
キーストローク・マウス操作の詳細ログ収集しない

⚠️ 金融・医療・保険・認証関連のサイトはデフォルトで記録対象外です。設定から追加の除外ドメインを指定することもできます。

4.データの保存場所

収集したすべてのデータは、ユーザーのブラウザ内の IndexedDB（dogear_db） にローカル保存されます。

  • データは Chrome のプロファイルフォルダ内にのみ存在します
  • 外部サーバー・クラウドへの送信・同期は行いません
  • 設定で最大保存件数を制限できます（デフォルト 10,000 件）
  • 設定ページの「すべて削除」ボタンでいつでも完全削除できます

5.データの利用目的

収集したデータは、以下の目的にのみ使用されます。

  • 履歴検索ページでのキーワード全文検索
  • データ分析ページでのブラウジングパターン可視化
  • ジャンル自動分類（ニュース・テクノロジー・ショッピングなど）
  • ユーザーによる JSON / CSV エクスポート

上記以外の目的でのデータ使用は行いません。

6.第三者への提供

✓ 収集したデータを第三者（個人・企業・サーバーを問わず）に販売・提供・転送することは一切ありません。

  • 広告目的でのデータ利用・転送は行いません
  • 信用力の判定・融資目的での使用は行いません
  • Google のデベロッパー プログラム ポリシーを遵守します

7.除外サイト（デフォルト）

以下のカテゴリのサイトは、デフォルトで記録対象から除外されています。

  • 銀行・証券（smbc, mufg, mizuho, rakuten-bank など）
  • 決済サービス（paypal.com, stripe.com など）
  • 医療・病院・クリニック関連
  • 保険関連
  • マイナンバー関連

設定ページから追加の除外ドメインをいつでも指定できます。

8.ユーザーによるデータ管理

  • 閲覧・エクスポート：履歴検索・データ分析ページで確認。JSON / CSV でエクスポート可能
  • 個別削除：履歴検索ページの各アイテムの削除ボタンで1件ずつ削除可能
  • 全件削除：設定ページの「すべて削除」ボタンで IndexedDB を完全削除
  • 拡張機能の削除：Chrome から拡張機能をアンインストールすることで、すべての記録が消去されます
  • 除外ドメインの追加：設定ページで記録したくないドメインを随時追加可能

9.権限の使用目的

権限使用目的
history インストール時に既存の Chrome 閲覧履歴をローカルデータベースにインポートするため。取得データは端末内にのみ保存。
storage ユーザー設定（滞在時間しきい値・除外ドメインなど）を保存するため。
tabs ポップアップから履歴検索・データ分析・設定ページを開くため。
activeTab ポップアップ操作時に現在のタブ情報を参照するため。
<all_urls>（ホスト権限） すべての Web ページに対してコンテンツスクリプトを実行し、段落テキストを検出・記録するため。データは端末内にのみ保存。
